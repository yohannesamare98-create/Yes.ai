# Client Milestone 4 Test Checklist

1. Open the client dashboard while logged out.
2. Confirm these options appear: Google, email/password, magic link, forgot password, free trial.
3. Log in with an existing email/password account.
4. Log out and test Google login.
5. Enter an existing email and send a magic link.
6. Test forgot password and save a new password from the recovery link.
7. Create a new account from `signup.html` and confirm the setup wizard opens.
8. Confirm the dashboard status card appears and the technical RLS banner is gone.
9. Test on iPhone/Android width and verify there is no horizontal scrolling.
10. Open the setup wizard and verify it fills the mobile screen and buttons remain reachable.
