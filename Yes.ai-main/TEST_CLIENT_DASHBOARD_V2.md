# Client Dashboard V2 Test Checklist

1. Open the client dashboard while logged out. Only the login screen should appear.
2. Test Continue with Google.
3. Log out, then test email/password login.
4. Enter an email and test Magic Link.
5. Enter an email and test Forgot Password.
6. Log in with an unfinished account. The Setup Wizard should open automatically.
7. Tap View Setup on iPhone Safari. The wizard should open full screen.
8. Test Back, Continue, Close, and Finish Setup.
9. Confirm there is no horizontal scrolling on iPhone.
10. Confirm the dashboard status card and all tabs load on iPhone, iPad, and desktop.
