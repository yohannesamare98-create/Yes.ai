# Test checklist

1. Open the client dashboard while logged out: only the login screen should appear.
2. Test Continue with Google.
3. Test email/password login.
4. Test Magic Link and Forgot Password.
5. After login, confirm the login screen disappears completely.
6. For an unfinished account, confirm the Setup Wizard opens automatically.
7. Close the wizard, tap View Setup, and confirm it opens on iPhone.
8. Refresh during onboarding and confirm the saved step resumes.
9. Confirm there is no `config.offers.map is not a function` error.
10. Test the layout on iPhone, iPad, and desktop.
